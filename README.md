# Libvirt Web Manager

By Karel Cermak | [Karlosoft](https://karlosoft.com).

## About

- Experimental project to manage virtual machines using Libvirt and PHP in Browser.
- Developed for a school project.
- The project is **not** intended for production use and is not affiliated with Libvirt or Red Hat, Inc.

<br>

## Functionality

- Set connection URI.
- List all virtual machines.
- Start, stop, destroy, pause and resume virtual machines.
- List set CPU and memory.
- Hotplug and unplug drives.

<br>

## How to run
- Deploy this project on your web server. Recommended to use Apache and PHP 8.0 or higher.
- Make sure you have Libvirt installed and configured on your server ([Documentation](https://php.libvirt.org/compiling.html)).
- Change .env file.
